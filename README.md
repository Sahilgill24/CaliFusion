## Run locally 

1. First clone the repository.

2. Setup the local walrus server.
    ``` sh
    cd Server/server
    npm i 
    node walrus.js
    ```
3. Setup the walrus client.
    ```sh
    cd clientnode
    npm i 
    node model.js
    ```
4. Initialize the interface.
   ```sh
    pnpm i
    pnpm dev
   ```
